# 🏋️ IRONFORGE GYM

**Forge Your Limits** — A modern Flask-powered gym website designed for fitness brands, training centers, and premium gyms.

Built with **Python Flask**, this project showcases dynamic membership plans, trainer profiles, gym facilities, and a sleek multi-page responsive layout.

---

## ✨ Features

* Dynamic Flask routing
* Modern responsive UI
* Membership pricing section
* Trainer & coaching showcase
* Gym facilities overview
* Reusable data-driven structure
* Clean and scalable project architecture

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/ironforge-gym.git
```

Navigate into the project folder:

```bash
cd ironforge-gym
```

Install dependencies:

```bash
pip install flask
```

Run the application:

```bash
python app.py
```

Open in browser:

```bash
http://127.0.0.1:5000
```
